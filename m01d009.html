Desafio de navegação

O desafio dessa vez é criar navegação entre páginas. Você deve criar uma página simples CINZA, com 2 links. Ao clicar no primeiro link, o visitante será enviado para a página VERDE. Note que na página verde existe um link para voltar para a página CINZA.
Se o visitante clicar no segundo link da página CINZA, será desviado para a página AMARELA. Não se esqueça de colocar um link para que o visitante possa voltar para a página CINZA também.
