Desafio das Tags

Agora é a hora de mostrar o que você está aprendendo de HTML5. Esse desafio faz referência uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag utilizada para fazer isso.

Resolução do desafio

Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá ser utilizada em cada caso.

  • Títulos
  • Parágrafos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Citações simples
  • Citações completas
  • Listas ordenadas
  • Listas não-ordenadas
  • Texto pré-formatado
